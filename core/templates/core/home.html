A way to perform and teach basic operations from the pandas data analytics package.

Web Pandas is a website designed by Timothy Malahy. The site is entirely open sourced and accessible at github. The intent is to teach people about pandas, a popular data analytics python backed by the Python programming language. The website is written using the Django Web Framework written in Python as well as Javascript, HTML, and CSS.
